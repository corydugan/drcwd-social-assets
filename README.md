# drcwd-social-assets

Public host for drCWDugan social-media images so Buffer (MCP + API) can attach them by URL.
Buffer's API only accepts media by **public URL** (no local upload — see Buffer API roadmap),
so images posted via Buffer must live at a fetchable URL. This repo is that URL home.

- `workforce-iron/` — the 6-post "Workforce-Iron" IG carousel series (2026-06-19 build).

Raw URL pattern:
`https://raw.githubusercontent.com/corydugan/drcwd-social-assets/main/<path>`
